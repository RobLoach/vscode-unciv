# vscode-unciv

This [Visual Studio Code](https://code.visualstudio.com/) extension provides validation for Unciv mod files. It automatically checks your Unciv JSON syntax errors and common mistakes, helping you catch issues early in your mod development workflow.

## Installation

```sh
code --install-extension unciv
```
